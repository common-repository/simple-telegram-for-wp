Postet neue Artikel oder aktualisierte in einem Telegram-Kanal.


== Description ==

Mit Simple Telegram ist es auf einfache Art und Weise möglich, neue (oder auch aktualisierte) WordPress-Artikel automatisiert in einem Telegram-Kanal zu posten. Dieses Plugin implementiert aber nur das Post-Feature, einen Telegram-Bot und einen Telegram-Kanal müsst ihr manuell anlegen, was aber auch in wenigen Sekunden erledigt ist.

<i>Simple Telegram</i> basiert auf dem <a href="http://notifcaster.com" target="_blank">Notifcaster</a>-Service von Ameer Mousavi.

= Optionale Funktionen =

* Anzeige der Linkvorschau
* Posten des Artikelbildes
* Markieren von aktualisierten und erneut gesendeten Artikeln mit "UPDATE"
* Eigenes Nachrichtentemplate mit Variablen
* Markdown-Support für das Nachrichtentemplate

= Systemanforderungen =
* PHP ab 5.2.4
* WordPress ab 3.4

= Installation =
1. Downloadpaket entpacken
2. Den Plugin-Ordner nach "/wp-content/plugins/" hochladen
3. Das Plugin in WordPress aktivieren
4. In die "Simple Telegram"-Einstellungen gehen und den weiteren Schritten folgen


Einfach in den Plugin-Ordner von WordPress extrahieren und aktivieren. Die restlichen Schritte zur Erstellung des Telegram-Bots und Telegram-Kanals findet ihr an dieser Stelle.
